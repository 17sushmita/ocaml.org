A taste of OCaml

type 'a tree =
  Leaf of 'a  |  Node of 'a tree * 'a tree

let rec exists_leaf test = function
  | Leaf v -> test v
  | Node (left, right) ->
      exists_leaf test left
      || exists_leaf test right

let has_even_leaf tree =
  exists_leaf (fun n -> n mod 2 = 0) tree

The code above defines the type of a binary tree with leaves holding values of an arbitrary type, as reflected by the type parameter 'a. For example, Leaf 1 is of type int tree while Leaf "hello" is a string tree. The function exists_leaf runs through the tree to seek a leaf containing a value that satisfies test (a function accepting values of the same type 'a as the tree and returning a boolean). Then has_even_leaf passes to it the anonymous function fun n -> n mod 2 = 0 to check whether the tree contains a leaf holding an even integer (thus its argument tree has type int tree).
